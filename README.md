# 🎬 Advanced Telegram YouTube Downloader Bot

A professional, production-ready Telegram bot for downloading YouTube videos, audio, and thumbnails with advanced features.

## ✨ Features

- 📥 Download YouTube videos in any quality
- 🎵 Convert videos to MP3 (192kbps)
- 🖼 Download video thumbnails
- 🔍 Search videos by name
- ⚙️ Persistent user settings
- 🎛 Dynamic quality selection
- 🧹 Automatic file cleanup
- 📊 Real-time download progress
- 🔒 Document-only uploads (no compression)

## 📋 Requirements

- Python 3.10 or higher
- FFmpeg (must be installed on system)
- Telegram Bot Token
